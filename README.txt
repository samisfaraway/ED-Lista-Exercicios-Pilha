Esse projeto é uma implementação de uma pilha encadeada do tipo LIFO (Last In, First Out) em C#. 

Foi feito por Samira Acioli.

Aqui estão algumas coisas que implementei:

1. Adicionar novos itens no topo da pilha (`push`).
2. Remover o item que está no topo (`pop`).
3. Verificar se um item está na pilha (`consulta`).
4. Mostrar todos os itens na pilha (`imprimir`).
5. Contar quantos itens estão na pilha.
6. Contar quantos números ímpares estão na pilha.
7. Separar números positivos e negativos em pilhas diferentes.
8. Inverter a ordem das letras na pilha.
9. Verificar se a sequência de letras forma um palíndromo (como "R A D A R").
10. Transferir os itens de uma pilha para outra, mantendo a ordem.

Para rodar isso, abra o terminal na pasta do projeto e digite:

bash
dotnet run
